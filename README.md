# Presentation

Este repositorio contiene una página de presentación personal para GitHub Pages.

Archivos principales:

- `index.html`: página principal con tres secciones: Home, Proyectos y Estudios.
- `styles.css`: estilos responsivos y modernos.
- `script.js`: navegación suave y resaltado de la sección activa.

Cómo usarlo:

1. Reemplaza el contenido de `index.html` con tus datos reales del CV.
2. Ajusta los proyectos y estudios dentro de sus secciones.
3. Sube los cambios a GitHub para que se publique con GitHub Pages.

La estructura ya está lista para usarse en GitHub Pages desde la rama principal.
